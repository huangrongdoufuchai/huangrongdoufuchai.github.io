如何用hugo搭建个人博客（mac）

首先安装hugo brew install hugo 检查是否安装好 hugo version 新建一个个人站点 hugo new site quickstart（quickstart可以命名为自己的github.io） 进入目录 cd quickstart git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke “>> config.toml” 新建一个博文 hugo new posts/my-first-post.md hugo server -D 根据显示 就可以点击进入自己的站点了 进入 config.toml 可以修改语言和主题 主题可以去github下载 hugo quickstart界面也有很多主题推荐。 baseURL = “https://example.org/" languageCode = “en-us” title = “My New Hugo Site” theme = “ananke” 最后一句 hugo 然后修改draft:false 完成 — title: “My First Post” date: 2019-03-26T08:47:11+01:00 draft: true — 第二部 部署到github
